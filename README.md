# Camera_filters
El filter_Camera es una camara que interactua con botones del 1 - 6 proporcionando filtros.
Los filtros son:
ESC: Salir
1: Camara
2: filtro gris
3: filtro Canny
4: Filtro Sobel x
5: Filtro Sobel y
6: Filtro Laplacian
G: Guardar imagen
C: Guardar imagen
P: Pausa
